kova

Triangular arbitrage engine for cryptocurrency markets. Discovers three-leg cycles across exchange order books, evaluates profitability in real time, and executes with built-in risk management.

Tests

Async Pipeline

Tokio-powered WebSocket streams, detector, executor, and stats as independent concurrent tasks.

Risk Management

Kill switch on consecutive losses or session loss. Min profit, max trade size, slippage buffer.

Paper Trading

Dry-run mode with live data. Opportunities logged to JSONL, queryable with DuckDB.

Multi-Exchange

Generic Exchange trait with static dispatch. Binance and Paribu adapters included.

Auto Discovery

Builds adjacency graph from all symbols and finds every valid triangle path automatically.

Trade Journal

Every opportunity serialized to JSONL with outcome, profit, and leg details for post-session analysis.

discover

Find Triangles

Scan exchange symbols and list valid three-leg cycles for a start asset.

kova discover --start-asset USDT
dry-run

Paper Trade

Stream live data, detect opportunities, log theoretical trades.

kova dry-run --config kova.toml
run

Live Trading

Execute real trades through the full pipeline with risk management.

kova run --config kova.toml
analyze

Review Results

Parse the trade journal and compute win rate, PnL, and top paths.

kova analyze --journal kova-journal.jsonl
quick start
$ git clone https://github.com/selimozten/kova.git && cd kova
$ cargo build --release
$ cp kova.toml.example kova.toml  # set api_key + api_secret
$ ./target/release/kova dry-run
